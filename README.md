## 项目笔记
### 项目概述
#### 电商后台管理系统的技术选型
1. 前端项目技术栈
- Vue
- Vue-router
- Element-UI：前端UI组件库
- Axios 
- Echarts：获取图形报表
2. 后端项目技术栈
- Node.js
- Express
- Jwt：状态保持工具 模拟类似于session的登录记录功能
- Mysql
- Sequelize：操控数据库的框架
